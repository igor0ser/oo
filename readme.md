To run SPA:

1. install bower: 

	> npm install -g bower

2. install npm dependencies:

	> npm install

3. install bower dependencies:

	> bower install

4a. To start a project use default gulp task

	> gulp


4b. To start tests use next command:

	> gulp test

5. Enjoy.
